## 一、执行摘要

### 核心发现

张雪峰现象是中国社会转型期多重结构性矛盾交汇的产物：高等教育大众化与就业市场紧缩之间的张力、教育公共信息服务的缺位、阶层流动通道收窄引发的集体焦虑，以及互联网时代个人IP对公共议题的话语权重构。张雪峰以"七分钟解读34所985高校"走红，将高考志愿填报从"信息盲区"推向"全民议题"，其话语体系的核心逻辑可概括为**"专业即就业工具，教育即阶层跃迁的投资回报"**。

**五项核心发现**：

1. **信息平权的悖论**：张雪峰确实打破了高校招生信息的不对称格局，但其商业化操作（梦想卡8999-12999元、圆梦卡17999-18999元）将公共信息转化为私有商品，形成新的信息付费壁垒。

2. **焦虑的生产与再生产**：其"寒门避险"叙事既是真实社会焦虑的镜像，也是焦虑变现的商业引擎。3小时营收2亿的传闻虽被否认，但峰学蔚来的多层产品矩阵确实实现了焦虑的系统性货币化。

3. **羊群效应与专业生态失衡**：张雪峰推荐的"烟草、电力、计算机"等专业分数线异常飙升，而生化环材等基础学科、新闻学等人文学科遭遇报考寒冬，引发人才结构的隐性危机。

4. **制度缺位的民间补丁**：高考志愿填报制度的信息透明度不足、专业设置与产业需求脱节、就业市场信号失真等结构性问题，是张雪峰现象得以生长的制度土壤。

5. **后张雪峰时代的真空**：2025年9月账号被封禁、2026年3月猝然离世后，峰学蔚来直播间在线人数从十几万骤降至不足3万，弹幕满屏"退钱"，暴露出个人IP驱动型教育咨询的根本脆弱性。

### 政策建议摘要

- **建立国家级升学指导公共服务平台**，整合专业就业数据、行业发展预测、毕业生跟踪调查等信息，免费向考生和家长开放
- **完善专业动态调整机制**，缩短专业设置与产业需求的响应周期至2-3年
- **推动升学指导职业化**，在学校体系内设置持证升学规划师，替代市场化焦虑消费
- **加强教育咨询行业监管**，明确服务标准、收费上限与信息披露义务
- **引导教育叙事多元化**，在主流媒体中拓展"成功"的定义维度，消解"唯就业论"的话语垄断

## 二、现象学描述：张雪峰话语体系的内容分析

### 2.1 人物谱系与走红路径

张雪峰，本名张子彪，1984年出生于黑龙江省齐齐哈尔市。2016年，一段"七分钟解读34所985高校"的视频在网络上爆红，标志着他从考研辅导教师向公众人物的转型。此后，他凭借"单口相声式"的升学指导风格，在微博、抖音、B站等平台积累了数千万粉丝，成为中文互联网上最具影响力的教育类IP之一。

2021年，张雪峰离开奋斗14年的北京，将公司迁至苏州，成立苏州峰学蔚来教育科技有限公司，核心业务从考研辅导转向高考志愿填报。这一转型精准切入了一个市场规模达数十亿元、供需严重失衡的新兴市场——据艾媒咨询数据，2023年中国高考志愿填报市场付费规模已达9.5亿元，且仍在高速增长。

**关键时间线**：

| 时间          | 事件                                                   |
| ------------- | ------------------------------------------------------ |
| 2016年        | "七分钟解读34所985"视频走红                            |
| 2021年        | 离京赴苏，成立峰学蔚来                                 |
| 2023年6月     | "学新闻就打晕"言论引发全国争议                         |
| 2023年12月    | "文科都是服务业，总结成一个字就是'舔'"言论再次引爆舆论 |
| 2024年7月     | "3小时入账2亿"传闻被否认，但产品售罄                   |

| 2025年10月    | 被官方定性为"教育功利化言论"                           |
| 2026年3月24日 | 因心源性猝死在苏州逝世，享年42岁                       |
| 2026年3月28日 | 追悼会在苏州殡仪馆举行                                 |

### 2.2 核心话语体系解构

张雪峰的公共话语可归纳为以下几个相互关联的论题集群：

#### 论题一："专业即就业工具"

> "我从来不跟学生谈理想，理想能当饭吃吗？你先活下来再说。"

这是张雪峰话语体系的基石论题。其论证结构为：

**前提1**：大学教育的首要功能是获得就业竞争力
**前提2**：不同专业的就业前景差异巨大
**前提3**：多数家庭（尤其是普通家庭）承受不起"选错专业"的代价
**结论**：选专业应以就业回报率为首要标准

这一论证的价值预设是：**教育的工具理性应优先于价值理性**。它回避了"教育何为"这一根本性问题，将大学矮化为职业培训所。

#### 论题二："文科服务业论"

> "所有的文科专业都叫服务业……总结成一个字儿，就是'舔'。"

这一论题以极度简化的二元框架（理科=创造价值/文科=服务他人）消解了人文社会科学的知识价值。其传播策略在于：用粗鄙化的语言制造情绪冲击力（"舔"字的使用），将复杂的专业选择问题简化为好懂好记的口号。

从论证角度看，这一论题存在严重的范畴混淆：将"服务业"等同于"低价值"，忽视了高端服务业（法律、金融、咨询、文化创意）的高收入特征，也无视了文科教育在批判性思维、文化传承、社会治理中的不可替代性。

#### 论题三："寒门避险逻辑"

> "普通家庭的孩子，别谈什么兴趣，先保证能找到工作。"

这一论题具有最强的社会共鸣力，因为它触及了阶层流动焦虑这一当代中国最敏感的社会神经。其叙事策略是将自己建构为"寒门代言人"——尽管他本人实际上是从体制内家庭走出的成功企业家。

"避险逻辑"的隐含前提是：教育资源是一种有限的、需要精算的投资品，而非一种权利或成长机会。这种框架将学生从"教育主体"降格为"投资客体"。

#### 论题四："天坑专业警示"

> "生化环材，四大天坑。土木机械，坑中之坑。"

张雪峰对特定专业的"劝退"话语，以高度标签化的方式重塑了公众对专业价值的认知。这些标签虽然包含真实的就业市场信息，但其极端化表述（如将整个学科领域一概否定）忽视了行业周期性变化、个体差异以及基础学科对国家创新体系的战略价值。

值得注意的是，土木工程专业恰恰是一个"预言自我实现"的经典案例：2016年前后张雪峰曾推荐该专业，而到2020年代随着房地产下行，该专业被他列入"劝退名单"——这暴露了基于当下就业数据做长期预测的"刻舟求剑"风险。

#### 论题五："打晕论"

> "孩子非要报新闻学，我一定会把他打晕！"

2023年6月的这一言论是张雪峰最出圈的公共事件，引发了重庆大学新闻学院教授张小强等多位学者的公开反驳。"打晕论"的修辞策略在于：以肢体暴力的夸张意象（打晕自己的孩子）来表达专业判断的极端确定性，从而制造戏剧化的情绪冲击。

这一事件的意义超越了新闻学专业的范畴，实质上是**"知识权威的代际更迭"**——一个网红用大众传播的话语权挑战了学院派的知识垄断。

### 2.3 传播机制分析

张雪峰的话语之所以比学院派论述更具传播力，源于以下几个机制的叠加：

1. **情绪优先于理性**：他的表达以情绪动员（焦虑、愤怒、解气）为核心，而非逻辑论证
2. **简化优先于复杂**：将多维问题压缩为二元对立（好专业vs坏专业、文科vs理科）
3. **故事优先于数据**：用"我认识一个学新闻的，毕业就失业"等个案代替统计分析
4. **冲突优先于共识**：主动制造与教授、专家的对立，将自己塑造为"说真话的人"
5. **平台算法共振**：短视频平台的推荐机制天然青睐争议性、情绪化的短句式表达

## 三、维度A：教育观念层——功利主义与理想主义的张力

### 3.1 两种教育哲学的对峙

张雪峰现象折射出当代中国教育领域最根本的价值冲突：**教育是"成人的事业"还是"谋生的工具"？**

**理想主义教育观**的代表可追溯至蔡元培的"教育者，养成人格之事业也"，以及西方博雅教育（Liberal Arts）传统中"教育的目的是培养自由而完整的人"的理念。这一传统强调：

- 教育的内在价值（intrinsic value）：知识本身就是好的，不需要外部目的来证明
- "无用之用"：看似无直接经济回报的人文素养，实为创新能力与生活智慧的根基
- 全人发展：教育应关注智识、情感、道德、审美等多维度成长

**功利主义教育观**的根源可追溯至明清科举制度中的"学而优则仕"传统，以及当代市场经济逻辑对教育领域的渗透。其核心主张为：

- 教育的工具价值（instrumental value）：教育投资应有可量化的经济回报
- 生存优先：在就业市场紧缩的现实下，"先活下来"是第一要务
- ROI最大化：选专业如同投资决策，应追求回报率最大化

**张雪峰的位置**：他并非功利主义教育观的发明者，而是其最有效的传播者和商业化实践者。他将一种弥漫性的社会心态提炼为清晰可操作的行动指南，并通过互联网放大为全民共识。

### 3.2 阶层分化的评价光谱

关于"普通家庭72%支持/高知家庭11%支持"这一数据（该数据来自网络讨论中的自我报告式统计，精确性有待核实，但定性趋势有广泛佐证），其背后的逻辑是：

**工薪阶层/普通家庭的支持逻辑**：

- 信息不对称的受害者：缺乏高校招生、专业就业的系统信息，张雪峰填补了这一真空
- 风险厌恶的理性选择：一个家庭的教育投资往往是最大的一笔支出，"选错专业"的代价可能是灾难性的
- 对体制内话语的不信任：对高校招生宣传、教授推荐的"正能量话语"持怀疑态度，认为张雪峰"说了真话"

**高知/精英家庭的反对逻辑**：

- 信息优势的自信：本身掌握足够的教育信息，不需要外部指导
- 对人文价值的坚守：多数高知家庭自身受益于人文教育，认为"唯就业论"是短视的
- 对话语暴力的反感：张雪峰"打晕""舔"等粗鄙化表达与精英审美格格不入
- 阶层焦虑的反向表现：高知家庭同样焦虑，但焦虑的是"阶层维系"而非"阶层跃迁"

### 3.3 "主体性消解"的批判

张雪峰话语体系中最深层的问题，不在于他推荐了哪些专业，而在于**他从根本上取消了学生作为教育主体的地位**。

在他的框架中：

- 学生不是"探索自我、发现兴趣的主体"，而是"需要被安排的客体"
- 专业选择不是"自我实现的过程"，而是"外部专家代为决策的结果"
- 教育不是"打开可能性的空间"，而是"关闭可能性以降低风险的收缩"

这种**主体性消解**的长期后果是：

1. **内在动机的萎缩**：被安排进入"好就业"专业但缺乏兴趣的学生，可能出现学业倦怠、职业迷茫
2. **适应性风险**：当下的"好专业"在四年后可能不再热门（土木工程的反转案例），缺乏内在驱动力的个体更难转型
3. **人文素养的空心化**：如果几代人都按"避坑指南"选择专业，基础学科和人文社科的人才断层将削弱国家的创新底蕴和文化软实力

### 3.4 回答核心问题

**教育本质究竟是"人的全面发展"还是"社会分层工具"？**

答案是：两者皆是，且不可偏废。教育确实承担着社会分层和劳动力配置的功能，但这不意味着应以此为唯一目标。正如一枚硬币的两面，工具理性和价值理性需要在具体的教育实践中寻求动态平衡。

**张雪峰是"戳穿幻象的启蒙者"还是"加剧异化的推手"？**

两者皆是，取决于观察的层面和时间段。在短期和微观层面，他的信息打破了部分家庭面临的信息盲区，具有启蒙意义。但在长期和宏观层面，他将一种原本应被质疑的功利主义逻辑推向极致，加剧了教育的异化——教育不再服务于人的发展，而是服务于不可控的就业市场波动。

## 四、维度B：市场机制层——教育咨询产业的异化与重构

### 4.1 商业模式解构

峰学蔚来的商业模型是一个以个人IP为核心、多层次产品矩阵为支撑、焦虑情绪为动力的精密系统：

**产品层级**：

| 产品名称   | 价格区间      | 内容描述                         |
| ---------- | ------------- | -------------------------------- |
| 免费内容   | 0元           | 短视频、直播片段（获客漏斗入口） |
| 低价引流课 | 数百元        | 基础志愿填报知识、专业解读       |
| 梦想卡     | 8999-12999元  | 一对一志愿填报规划               |
| 圆梦卡     | 17999-18999元 | 高端定制化升学规划               |

**商业逻辑**：

1. **焦虑获客**：免费短视频和直播制造焦虑（"你不知道这些就毁了孩子一生"）
2. **信任转化**：通过"敢说真话""为普通人代言"的人设建立信任
3. **分级变现**：从免费到18999元的多层价格阶梯，覆盖不同支付能力的客群
4. **饥饿营销**：高端产品限量发售、限时抢购，制造稀缺感

**营收规模**：虽然"3小时入账2亿"被张雪峰本人否认，但其商业帝国的实际规模依然可观。考虑到千万级粉丝基数、高端产品的定价以及高考季的集中消费特征，峰学蔚来的年营收体量应达数亿元级别。

### 4.2 信息差经济学：公共信息私有化的伦理困境

张雪峰商业模式的底层逻辑是**信息差套利**：

- **信息供给侧**：高校招生数据、专业就业率、行业薪资水平等本应是公共信息
- **信息需求侧**：1300万高考生及其家庭需要这些信息来做重大人生决策
- **信息中介**：张雪峰团队将散落的公共信息整合、解读、包装为付费产品

**积极意义**：

- 确实降低了部分家庭获取信息的成本（相比于自己去翻几百所高校的招生简章）
- 以市场力量倒逼高校提高招生信息的透明度和可读性
- 为高考志愿填报行业建立了产品标准和服务规范

**伦理争议**：

- 利用公共信息牟利是否合理？这些数据本应由教育部门免费、系统地提供给所有考生
- 高价咨询是否制造了新的教育不平等？能支付18999元咨询费的家庭与无法支付的家庭之间，信息鸿沟是否反而扩大了？
- 当公共信息被私人垄断并商品化，"教育公平"的承诺如何兑现？

### 4.3 "张雪峰效应"：羊群效应与专业生态

张雪峰的推荐对高校专业报考产生了显著的**羊群效应**（Herding Effect）：

**典型案例**：

- **烟草类专业**：经张雪峰推荐后，部分院校烟草相关专业的录取分数线飙升，出现双非院校烟草专业反超部分985院校同分段专业的异常现象
- **电力类专业**：华北电力大学等电力院校的报考热度大幅上升，录取分数线逐年走高
- **计算机/人工智能**：虽然本身热门，但张雪峰的反复推荐进一步加剧了扎堆效应
- **新闻学**：2023年"打晕论"后，部分院校新闻学专业报考人数和录取分数线出现明显下降
- **生化环材**：被贴上"天坑"标签后，相关专业在部分院校面临招生困难

**结构性后果**：

1. **热门专业过热**：大量考生涌入少数"推荐专业"，四年后可能面临就业市场饱和
2. **冷门专业过冷**：基础学科和人文社科的人才供给萎缩，影响国家长远创新能力和文化传承
3. **高校的应激反应**：部分高校开始主动调整专业设置以迎合"市场偏好"，2025年教育部推动裁撤低就业率专业，150余所高校公示撤销汽车服务工程（9个撤销点）、市场营销（7个）等专业

### 4.4 阶层再生产悖论

一个核心悖论在于：张雪峰自诩为"寒门代言人"，但其商业模式的受益者是否真的是寒门？

**论点一：高价咨询的阶层筛选效应**

- 18999元的"圆梦卡"对于月收入5000元的工薪家庭而言，相当于近4个月的收入
- 真正的"寒门"可能连免费直播都无法看到（需要智能手机、网络、以及获取信息的时间成本）
- 实际购买高端产品的客户画像可能更接近中产而非底层

**论点二：信息付费壁垒**

- 当越来越多的家庭依赖付费咨询来做决策，那些无法支付费用的家庭面临的信息劣势进一步扩大
- 这创造了一种新型的教育不平等：不是"有没有学上"的不平等，而是"有没有好的信息来做选择"的不平等

**论点三：公益人设与资本逻辑的张力**

- 张雪峰确有公益行为（如资助贫困生），但这与其商业模式之间存在结构性矛盾
- 公益行为强化了"好人设"，进而推动了更多商业转化——公益成为商业的营销工具

## 五、维度C：升学制度层——制度缺陷与民间应对的互动

### 5.1 制度缝隙诊断

张雪峰现象之所以能够生长，根源在于中国升学制度存在系统性的信息与结构缺陷：

**缺陷一：招生信息的碎片化与不透明**

- 各高校的招生简章、专业介绍、就业数据分散在不同渠道，格式不统一，可读性差
- 部分高校的就业率数据存在注水嫌疑，可信度存疑
- 缺乏全国统一的、用户友好的志愿填报信息平台

**缺陷二：专业设置与产业需求的结构性脱节**

- 高校专业的设置调整周期通常为4-5年（一个本科学制），远慢于产业变化速度
- 部分专业（如公共事业管理、信息管理与信息系统）的课程内容严重滞后于行业发展
- 2026年2月教育部发布《关于深化职业教育教学关键要素改革的意见》，明确提出"动态调整专业设置，裁撤过剩专业"，显示决策层已意识到问题

**缺陷三：就业市场信号的失真**

- 大学毕业生就业率统计口径不一，存在"被就业"现象
- 专业对口率、薪资中位数、职业发展轨迹等关键数据缺乏公开透明的统计发布
- 考生和家长面对的是一个信息极度不对称的市场

**缺陷四：学校升学指导的缺位**

- 绝大多数高中缺乏专职的升学规划师
- 高中教师的升学指导往往局限于"冲稳保"的分数策略，缺乏对专业、行业、职业的系统了解
- 学校升学指导服务的缺位，为市场化咨询机构提供了巨大的市场空间

### 5.2 民间策略的制度化倒逼

张雪峰推荐的若干策略，客观上对制度改革产生了倒逼效应：

**"理科重专业/文科重学校"**

- 这一策略反映了当前就业市场对理工科专业技能的高度需求，以及对文科毕业生"通用能力"的认可
- 倒逼部分高校加强文科教育的差异化和专业化

**"避天坑保就业"**

- 加速了教育部对低就业率专业的清理：2025年多个省份推动高校撤销低就业率专业点
- 促使高校反思专业设置的合理性，从"有什么老师就开什么专业"转向"市场需要什么就培养什么"

**"看就业数据选专业"**

- 推动教育部门和第三方机构加强就业数据的收集和发布
- 麦可思等第三方研究机构的毕业生跟踪调查报告影响力日增

### 5.3 预测性局限：刻舟求剑的风险

张雪峰模式最根本的缺陷在于其**预测性局限**：

- 高考志愿填报的决策时刻与就业市场兑现时刻之间存在4-7年的时间差
- 基于当下就业数据指导未来选择，本质上是一种"用后视镜开车"的行为
- 土木工程是最典型的反转案例：2016年前后被推荐为"铁饭碗"，到2020年代中期被列为"天坑"
- 今天的"热门专业"（如人工智能、大数据）是否会在四年后面临过饱和？

这并非否定就业信息的价值，而是指出：**任何基于当前数据的确定性判断都是可疑的，真正的教育决策应培养学生的适应性和学习能力，而非押注某个具体专业**。

### 5.4 回答核心问题

**张雪峰现象是制度失灵下的民间自救，还是破坏教育生态的扰动因素？**

答案是：**首先是制度失灵的症状，其次才成为扰动因素**。如果教育部门能提供透明、系统、免费的升学信息服务，如果学校能提供专业的升学指导，张雪峰的市场空间将大幅缩小。他的出现是结果，不是原因。

但当这种"民间自救"被商业化放大后，它确实产生了扰动效应：专业报考的羊群效应、人文精神的边缘化、教育焦虑的系统化再生产。这说明，单纯依靠市场机制来解决教育公共品的供给问题，会产生严重的外部性。

## 六、维度D：社会结构层——焦虑传导与阶层流动

### 6.1 焦虑社会学：时代症候的集中表达

张雪峰现象是当代中国社会焦虑的一面棱镜。以下几种焦虑在他的话语中得到了集中表达：

**就业焦虑**：

- 2024年高校毕业生人数达1179万，2025年预计超过1200万
- 青年失业率在2023年一度超过20%（后统计局暂停发布调整口径）
- "毕业即失业"从调侃变为部分群体的真实体验

**学历贬值焦虑**：

- 本科学历从"稀缺资源"变为"基本门槛"
- "双非"本科毕业生在就业市场上面临的歧视日益显性化
- 考研人数从2019年的290万飙升至2023年的474万，折射出"学历军备竞赛"

**阶层滑落恐惧**：

- 中产家庭最深层的恐惧是"阶层向下流动"
- 教育被视为阶层维系的最后防线
- "选错专业=阶层滑落"的等式被焦虑放大为社会共识

**张雪峰的角色**：他既是这些焦虑的镜像（真实反映了社会情绪），也是焦虑的放大器（通过话语策略将弥漫性焦虑转化为行动性恐慌）。他的商业成功证明了一个残酷的等式：**焦虑=注意力=金钱**。

### 6.2 "寒门叙事"的政治解构

张雪峰自我建构的"寒门代言人"身份，是一个精明的政治定位：

**叙事策略**：

- 以"普通家庭""寒门子弟"的话语框架，将自己置于道德制高点
- 将反对者（教授、学者）标签化为"何不食肉糜"的精英阶层
- 将专业选择问题转化为阶层正义问题：谁有权告诉"寒门"该学什么？

**身份矛盾**：

- 张雪峰本人出身于东北体制内家庭（父亲为公务员），虽非大富大贵，但并非严格意义上的"寒门"
- 他的商业模式（高价咨询、直播带货）使他成为身家数亿的企业家
- "寒门"叙事既是真诚的社会关怀，也是精准的品牌定位

**效用评估**：

- 对部分信息匮乏的家庭，他的建议确实提供了有价值的避坑指南
- 但"寒门叙事"的泛化使用，也可能制造一种误导：仿佛所有"普通家庭"都面临同样的困境和需求
- 阶层内部的多样性（不同地区、不同行业、不同期望）被"寒门vs精英"的二元框架所遮蔽

### 6.3 成功标准的单一化批判

张雪峰话语中最值得警惕的倾向，是**成功标准的极度单一化**：

在他的框架中，"好专业"的定义高度趋同：

- 考公/考编的"上岸"率
- 进入大厂/央企/国企的概率
- 起薪水平和薪资天花板
- 工作稳定性（"铁饭碗"）

这种单一化导向的深层问题：

1. **创新精神的压制**：如果所有人都涌向"稳定"的选项，谁来创业？谁来冒险？谁来做"不确定但可能改变世界"的事情？

2. **社会多元性的丧失**：一个健康的社会需要各种各样的人才——艺术家、社会工作者、基础研究者、公益从业者……当"好工作"被窄化为"体制内+高薪"，这些不可或缺的角色将面临人才断层

3. **个体幸福感的侵蚀**：大量研究表明，职业满意度与收入之间并非线性关系。被安排进入"高薪但无趣"的专业和职业的个体，可能面临更高的心理健康风险

4. **"读书无用论"的反面——"读书只为赚钱"**：张雪峰表面上反对"读书无用论"，但实际上强化了另一种更隐蔽的偏见——"读书的意义仅在于赚钱"。这与"读书无用论"看似对立，实则共享同一个价值前提：知识的唯一价值是经济回报

### 6.4 长期人才结构影响

**基础学科人才流失**：

- 物理、化学、生物、材料等基础学科的优秀生源持续流失
- 这些学科是国家科技创新体系的根基——芯片、新药、新材料的研发都依赖基础学科的长期积累
- 短期就业市场的"冷门"不代表长期战略价值的缺失

**人文社科边缘化**：

- 历史、哲学、文学、社会学等学科被视为"无用"
- 但这些学科培养的批判性思维、文化理解力、社会治理能力，是任何技术都无法替代的
- 一个只有工程师而没有思想家的社会，技术创新可能失去方向感和人文温度

**潜在的国家竞争力风险**：

- 如果"张雪峰效应"持续数年，中国可能面临基础研究人才断层和文化软实力下降的双重危机
- 这种影响是滞后的、隐性的，但一旦形成，修复周期将以十年计

## 七、辩证评估矩阵

### 7.1 利弊权衡

| 维度         | 积极面                 | 消极面                       | 条件限定                     |
| ------------ | ---------------------- | ---------------------------- | ---------------------------- |
| **信息公平** | 打破了部分信息不对称   | 将公共信息私有化，制造新壁垒 | 取决于公共信息服务的跟进速度 |
| **教育选择** | 提供了务实的避坑指南   | 窄化了选择空间，消解了主体性 | 对信息匮乏家庭的边际效用更大 |
| **专业生态** | 倒逼低就业率专业改革   | 引发羊群效应，损害基础学科   | 需配合专业动态调整机制       |
| **市场生态** | 推动教育咨询行业规范化 | 焦虑变现、饥饿营销的伦理问题 | 需加强行业监管               |
| **社会心态** | 倡导务实理性           | 加剧焦虑、单一化成功标准     | 需多元叙事的平衡             |
| **制度改革** | 倒逼信息公开和专业调整 | 以市场逻辑替代公共治理       | 制度回应需系统性设计         |
| **阶层流动** | 为部分家庭提供避险策略 | 高价服务制造新的不平等       | 取决于公共指导服务的覆盖范围 |

### 7.2 核心判断

张雪峰现象的总体评估不能简单地以"利大于弊"或"弊大于利"来概括，因为它在不同维度、不同时间段、不同社会群体中产生了截然不同的效应：

- **对信息匮乏的工薪家庭**：短期利大于弊，提供了有价值的避坑信息
- **对人文社科和基础学科**：弊大于利，加剧了人才流失和学科边缘化
- **对教育公共治理**：既是倒逼改革的外部压力，也是市场化替代公共品的负面示范
- **对社会心态**：弊大于利，加剧了焦虑传播和成功标准单一化

## 八、国际比较：升学指导制度的他山之石

### 8.1 美国：职业咨询的专业化体系

美国拥有全球最成熟的学校升学指导体系：

- **学校心理咨询师制度**：多数高中配备专职的School Counselor，为学生提供个性化的升学和职业规划指导
- **全国学生辅导师协会（ASCA）**：制定了系统的行业标准和伦理规范
- **大学招生信息透明度**：各大学必须向教育部提交详细的招生数据、毕业率、就业情况，通过College Scorecard等平台向公众免费开放
- **职业兴趣测评工具**：Holland职业兴趣量表、MBTI等工具被广泛应用于学校升学指导

**对中国的启示**：将升学指导从市场化服务转变为公共教育服务的组成部分，是解决"张雪峰困境"的根本路径。

### 8.2 德国：双元制教育的职业分流

德国的教育体系通过早期分流（通常在10-12岁）和双元制职业教育，实现了教育与就业市场的高效对接：

- **职业预科（Hauptschule）、实科中学（Realschule）、文理中学（Gymnasium）**的三级分流
- **双元制**：学生在企业实践和学校理论学习之间交替，毕业即具备就业能力
- **职业资格认证体系**：约350种国家认证的职业资格，为不同技能水平提供清晰的职业路径

**对中国的启示**：职业教育的社会地位提升和与普通教育的等值化，可以从根本上缓解"唯大学论"和"唯专业论"的焦虑。

### 8.3 日本："就活"文化与就业指导

日本的"就活"（就職活動，求职活动）文化是一个可供比较的镜像：

- 大学设有专门的"就职课"，提供从简历修改到面试模拟的全套服务
- 企业招聘有全国统一的时间表（虽然近年来有所松动）
- "就活"文化同样存在过度焦虑、形式主义等问题，但至少将就业指导制度化为大学教育的一部分

**对中国的启示**：将就业指导嵌入大学教育体系，而非留给市场化的"志愿填报"咨询，可以前置性地缓解焦虑。

### 8.4 国际坐标的定位

从国际比较来看，中国升学指导制度的突出特征是：

1. **公共服务缺位**：缺乏像美国ASCA那样的学校专职升学指导体系
2. **信息透明度不足**：没有像College Scorecard那样统一、免费、权威的信息平台
3. **市场化替代过早**：在公共制度尚未建立的情况下，市场化咨询机构过快填补了空白
4. **职业教育边缘化**：不像德国那样有成熟的职业分流和双元制体系

张雪峰现象在某种意义上是这些制度缺失的"代偿反应"——当公共制度无法满足需求时，市场和个人IP填补了真空。

## 九、社会启示与政策建议

### 9.1 短期措施（1-2年）

**建议一：建立国家级升学信息公共服务平台**

- 整合教育部、人社部、各高校的数据资源，建设一个统一的、免费的升学信息平台
- 平台应包含：各专业就业率（分学历层次）、薪资中位数和分布、行业发展趋势、毕业生满意度等
- 参考美国College Scorecard的设计，确保数据的标准化、可比性和用户友好性

**建议二：规范教育咨询行业**

- 明确教育咨询机构的准入门槛、服务标准和信息披露义务
- 对收费设置合理上限，防止过度商业化
- 要求咨询机构披露其预测方法论和历史准确率

**建议三：加强对"焦虑营销"的监管**

- 对利用教育焦虑进行商业推广的行为进行规范
- 平台应承担内容审核责任，对极端化、误导性的升学建议进行标注或限流

### 9.2 中期措施（2-5年）

**建议四：在学校体系内建立升学指导制度**

- 在高中和大学阶段设置专职升学规划师岗位
- 参考美国ASCA模式，制定行业标准和培训体系
- 将升学指导纳入教师培训和学校评估体系

**建议五：完善专业动态调整机制**

- 缩短专业设置与产业需求的响应周期至2-3年
- 建立"专业预警"制度：当某专业连续多年就业率低于阈值时，自动触发审查和调整机制
- 但同时设置"战略保护"清单：基础学科和人文社科即使就业率偏低，也应获得国家层面的保护性支持

**建议六：提升职业教育的社会地位**

- 通过政策引导（如提高技术工人薪资待遇、完善职业资格认证）改变"唯大学论"的社会认知
- 扩大"职教高考"的覆盖范围和认可度
- 引入德国双元制的有益经验，加强校企合作

### 9.3 长期措施（5-10年）

**建议七：引导教育叙事多元化**

- 在主流媒体和公共话语中拓展"成功"的定义维度
- 鼓励各行各业的优秀代表讲述多元化的职业故事
- 在中小学教育中融入生涯规划和自我认知课程

**建议八：构建终身学习体系**

- 在技术变革加速的时代，"一次选择定终身"的模式已不合时宜
- 发展微证书、继续教育、职业转型培训等灵活的学习路径
- 从根本上降低"选错专业"的代价——如果终身学习体系足够完善，任何一次选择都不是"终局"

**建议九：加强教育研究与循证决策**

- 支持对教育回报率、专业选择与职业发展的长期跟踪研究
- 用数据而非直觉来指导教育政策
- 建立毕业生10年、20年职业发展追踪数据库

## 十、理论提升：从个案到"教育功利主义普遍化"的范式讨论

### 10.1 张雪峰现象的理论定位

张雪峰现象不应被简单理解为"一个网红说了些有争议的话"，而应被视为**教育功利主义在数字时代的系统性表达**。

从社会学视角看，它是以下几种结构性趋势的交汇点：

1. **高等教育大众化**：中国高等教育毛入学率从1998年的9.8%飙升至2023年的60%以上。当大学不再是少数人的特权，"上大学"的意义就从"成才"转变为"就业"。

2. **就业市场的结构性紧缩**：经济增长减速、产业结构调整、技术替代加速，使得"好工作"的供给增速远低于大学毕业生的增速。

3. **互联网对知识权威的解构**：张雪峰用短视频和直播取代了教授和专家，用"段子"取代了学术话语。这是整个社会知识生产与传播方式变革的缩影。

4. **教育市场化与资本逻辑的渗透**：当教育被视为"投资"，回报率自然成为核心指标。张雪峰的商业模式只是将这种逻辑推向了极致。

### 10.2 "教育功利主义"的三重悖论

**悖论一：理性选择的非理性后果**

- 当每个个体都按"最优就业回报"选择专业时，集体行动的结果是热门专业过饱和、冷门专业断层
- 这是一个经典的"公地悲剧"：个体理性导致集体非理性

**悖论二：信息解放与信息异化**

- 更多的就业信息本应帮助人们做出更好的选择
- 但当信息被过度简化、情绪化传播后，它反而可能制造新的认知偏差
- "知道所有专业的就业率"不等于"理解什么是适合自己的"

**悖论三：阶层突围与阶层固化**

- 张雪峰的初衷（或至少其公共叙事）是帮助"寒门"实现在教育赛道上的突围
- 但如果所有人都按同一种策略行动，"突围"就变成了"内卷"
- 更根本的是，当教育被窄化为就业工具，它作为阶层流动通道的功能本身就被削弱了——因为真正决定阶层的，从来不只是专业选择

### 10.3 后张雪峰时代的范式转换

张雪峰的离世（2026年3月24日）和此前的账号封禁（2025年9月），标志着"个人IP驱动型教育咨询"范式的危机。峰学蔚来在他去世后的表现——直播间人数从十几万降至不足3万、弹幕满屏"退钱"——暴露了这一模式的根本脆弱性：**当整个商业体系锚定于一个人的IP，这个体系的存续就与这个人的生命等长**。

这为"后张雪峰时代"提出了根本性的问题：

1. **谁来接棒？** 教育功利主义的需求端（焦虑的家长和考生）并没有因为张雪峰的离去而消失。如果公共服务不能填补真空，新的"张雪峰"迟早会出现。

2. **制度化替代个人化**：将升学指导从个人IP模式转变为制度化的公共服务，是避免"一个张雪峰倒下去、千万个焦虑贩卖机站起来"的关键。

3. **价值重建**：在功利主义和理想主义之间寻找新的平衡点，需要整个社会——不仅是教育系统——对"什么是好的生活"进行更深层的反思。

### 10.4 终极之问

张雪峰现象最终指向一个古老的哲学问题：**教育的目的是什么？**

亚里士多德区分了"为了闲暇的教育"（liberal education）和"为了必需的教育"（vocational education）。前者旨在培养自由人，后者旨在训练劳动者。两千多年来，这两种教育观的张力从未消解。

在当代中国语境下，这一张力被以下现实急剧放大：

- 数以百万计的大学毕业生面临就业困境
- 家庭对教育的经济投入达到历史最高水平
- 社会流动性下降引发的集体焦虑
- 技术变革使得"终身一技"不再可能

在这种背景下，纯粹的理想主义教育观（"追求知识本身的价值"）确实显得不合时宜。但如果完全倒向功利主义（"只看就业回报"），教育将丧失其最根本的使命：**帮助每个人成为他自己**。

或许答案不在于选择哪一端，而在于重建一种更包容的教育叙事：承认就业是教育的重要目标之一，但不是唯一目标；承认不同家庭有不同的风险承受能力，因此需要差异化的指导；承认教育的终极价值不在于"找到好工作"，而在于"成为一个完整的人"——而"找到好工作"只是这个完整人生的一部分。

## 十一、结语

张雪峰的一生（1984-2026）浓缩了当代中国教育焦虑的诸多面向：从东北小城到北京打拼的"北漂"经历，从考研辅导教师到教育企业家的商业跃迁，从"说真话的网红"到"被封禁的争议人物"的命运转折，直至42岁因心源性猝死的突然告别。

他的遗产是复杂的：

- **他打破了一些幻象**，但也制造了新的幻象
- **他帮助了一些家庭**，但也可能误导了一些家庭
- **他倒逼了一些改革**，但也加剧了一些失衡
- **他说了很多真话**，但真话不等于全部的真相

对他最公允的评价或许是：**他是一个时代症候的集中表达者，而非这个症候的病因。** 真正需要反思的，不是"为什么会有张雪峰"，而是"为什么我们这个社会如此需要张雪峰"。

当教育公共服务足够完善、信息足够透明、选择足够多元时，"张雪峰"这个角色就不那么不可或缺了。这或许才是他最希望看到的——尽管，那也将意味着他自己不再被需要。

"后张雪峰时代"的真正课题，不是寻找下一个张雪峰，而是建设一个不再需要张雪峰的教育制度。
